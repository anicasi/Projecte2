# Projecte2
En aquesta pràctica es treballa amb entrades analògiques de l’ESP32, que permeten llegir valors continus provinents de sensors. El sistema ha de:
Llegir la lluminositat amb un LDR.
Llegir la temperatura amb un LM35.
Activar alertes automàtiques quan es superen llindars.
Respondre a comandes com STATUS i LED_OFF.
Mostrar les dades en temps real pel Monitor Sèrie.

En conclusió: Crear un sistema que vigili les condicions d'una sala de servidors. El sistema ha d'automatitzar alertes basades en llindars de temperatura i llum, a més de permetre la consulta de dades en temps real via Serial.
